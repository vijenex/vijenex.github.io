# Security Policy

## Brand Protection

**Vijenex™** is a registered trademark (Application No. 7368905, Class 9, India).

### Prohibited Activities

❌ **DO NOT:**
- Clone this website and rebrand it as your own
- Use the Vijenex™ name or logo without permission
- Create fake/phishing websites impersonating Vijenex™
- Modify and redistribute as a competing service
- Remove trademark notices or copyright information

### Allowed Usage

✅ **YOU MAY:**
- Fork the code for learning purposes (with attribution)
- Use the code structure for your own projects (with different branding)
- Contribute improvements via pull requests
- Report security issues responsibly

## Reporting Security Issues

If you discover a security vulnerability or someone impersonating Vijenex™:

1. **DO NOT** open a public issue
2. Report privately via GitHub Security Advisories
3. Or create a private issue in our repository
4. Include: description, steps to reproduce, impact assessment

We will respond within 48 hours.

## Legal Action

We actively monitor for:
- Trademark infringement
- Phishing attempts
- Unauthorized use of brand assets
- Code theft and rebranding

Violations will result in:
- DMCA takedown notices
- Legal action under Indian trademark law
- Reporting to hosting providers and domain registrars

## Responsible Disclosure

We appreciate security researchers who:
- Report issues privately
- Allow reasonable time for fixes
- Do not exploit vulnerabilities

Acknowledged researchers will be credited (with permission).
